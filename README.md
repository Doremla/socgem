# socgem
Un'applicazione per la sicurezza dei tuoi dispositivi, adatta per piccole aziende, facile e intuitiva da usare

CARTELLA DATABASE: viene messo il file .sql per importare e esportare il database
finchè verrà usato in locale

CARTELLA WEBPAGE: vengono inseriti i file .html, .php, .js etc...

CARTELLA API: cartella per i file che gestiscono le api, verrà usato flask e jwt per gestione dei token
